Class LayerList

Eine Liste von Layern, immer alphabetisch nach Namen sortiert. Die Namen müssen eindeutig sein. Das Zufügen bzw. Umbenennen eines Layers, so dass Mehrdeutigkeiten entstehen würden, führt zu einer Exception.
