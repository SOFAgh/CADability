Class ColorList

Eine Tabelle von benannten Farben. Die Tabelle ist serialisierbar und wird gewöhnlich in einem "Settings" Objekt gespeichert. Die Tabelle kann in der EigenschaftenAnzeige interaktiv manipuliert werden.
