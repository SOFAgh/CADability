Class Picture

A bitmap as a rectangular or rhombiod face in 3D space. This object is mainly used for illustration or as a background in a model. It is a GeoObject and may be part of a model. It is defined by the bitmap bits and the location in space.

Usually it is the path and filename, especially when the object is beeing generated interactively by CADability's user interface. Setting the value does NOT load the specified bitmap. This must be done seperately by setting the Bitmap property
