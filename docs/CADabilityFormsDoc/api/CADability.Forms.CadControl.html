Class CadControl

This class is an UserControl which allow us to use the cad in a custom form. The idea is not to force the user to use a form derived from CadForm. If the CreateMainMenu property is true, it creates the MainMenu in the parent form. The ProgressAction property allow us to use a custom progress ui.
